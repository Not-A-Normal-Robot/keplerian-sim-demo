The current Y-velocity relative to the parent body (v.y).

    v.y = e_21 ∙ o′.p + e_22 ∙ o′.q
    e_21 = cos(ω) sin(Ω) + sin(ω) cos(i) cos(Ω)
    e_22 = cos(ω) cos(i) cos(Ω) - sin(ω) sin(Ω)

...where:
ω = argument of periapsis
Ω = longitude of ascending node
i = inclination
o′.p = PQW velocity's P-component
o′.q = PQW velocity's Q-component
