See minimum requirements

We require a somewhat modern browser with these following features:

  • JavaScript support
  • WebGL support
  • WebAssembly support

Opening the console

You can open the console with the following keybinds:

  • F12
  • Ctrl + Shift + K

On Firefox, you can try to see in the "Network" tab if a request failed. This is similar on Chrome.
If a request failed, try refreshing the page or trying again later.
If you see an error in the console tab, then that might indicate a problem in the software itself.
You can report it to the developer by submitting a bug report issue: https://github.com/Not-A-Normal-Robot/keplerian-sim-demo/issues/new

Opening the console is not very easy in mobile and is out of the scope of this guide.
